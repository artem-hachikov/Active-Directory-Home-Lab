# 04 – Account Management

## Objective
Perform Active Directory account lifecycle management tasks including expiration settings, disabling/enabling accounts, and logon hour restrictions.

---

## Steps Performed

### 1️⃣ Account Expiration Configuration
- Configured account expiration date
- Verified expiration settings in user properties

### 2️⃣ Account Disable / Enable
- Disabled user account
- Confirmed disabled status in Active Directory Users and Computers (ADUC)
- Re-enabled account successfully

### 3️⃣ Logon Hour Restrictions
- Configured allowed logon hours
- Tested login outside allowed time
- Verified access denied behavior

---

## Skills Practiced
- Account lifecycle management
- Security enforcement configuration
- User restriction implementation
- Real-world Help Desk troubleshooting
